# NoCode Fit

一个基于 React + Vite 的热量计算器应用。

## 功能特性

- 计算基础代谢率 (BMR)
- 计算每日能量消耗 (TDEE)
- 计算减脂能量摄入
- 响应式设计，支持移动端

## 技术栈

- React 18
- Vite
- Tailwind CSS
- React Hook Form
- Lucide React Icons

## 开发

```bash
# 安装依赖
yarn install

# 启动开发服务器
yarn dev

# 构建生产版本
yarn build
```
